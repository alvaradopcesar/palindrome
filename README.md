## Getting Started

This project uses the **Go** programming language (Golang)

### Prerequisites

[Go](https://golang.org/) at least in version 1.16

### Installing


```

#### Using GOMODULE

```bash
go mod download
```

## Running the tests

```bash
go test ./...
```

## Deployment

Clone the repository

```bash
git clone git@github.com:alvaradopcesar/palindrome.git
```

Enter the repository folder

```bash
cd palindrome 
```

Build the binary

```bash
make
```

Run the program

```bash
# In Unix-like OS
./app

# In Windows
app.exe
```
